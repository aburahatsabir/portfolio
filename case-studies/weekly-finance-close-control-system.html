Engineering Trust in
Weekly Finance

Moving beyond spreadsheets to a logic-gated system. How we eliminated version drift and reduced reporting risk by 100% through forced reconciliation checkpoints in Excel & VBA.

The Challenge

The "V2_Final_Real" Chaos

The finance team was operating on a monthly close cadence, but executives demanded weekly visibility. The result was a frantic Friday morning rush involving manual copy-pasting, broken formulas, and zero version control.

  • No Input Validation Users could overwrite formulas with hardcoded numbers.
  • Retroactive Fixes Reconciliations happened after reporting, eroding trust.

The Solution

The Logic Gate Architecture

We replaced the "honor system" with hard-coded logic gates. Data cannot physically proceed to the reporting stage without passing validation.

1

Standardization

Raw extracts are dropped into a directory. PowerQuery automatically cleans, formats, and flags data type errors immediately.

Control Point
2

Forced Validation

If Variance > 0 then Lock. The "Publish" ribbon is disabled via XML until variance = 0 or an explanation is logged.

3

Immutable Output

System hard-codes values into a PDF with timestamp watermarks and auto-archives to a read-only network directory.

Under the Hood

This interactive demo simulates the actual VBA logic deployed. The system hooks into the Workbook_BeforeSave event and custom Ribbon XML.

gate_logic.vba 
Function CheckGateStatus() As Boolean
    Dim cell As Range
    ' Loop through defined named range
    For Each cell In Range("Control_Range")

        ' Logic Gate: Variance exists
        If cell.Value <> 0 Then
            CheckGateStatus = False
            LockDashboard ' Disables Ribbon
            MsgBox "Variance detected at " & cell.Address
            Exit Function
        End If
    Next cell

    CheckGateStatus = True
    UnlockExport ' Enables PDF Button
End Function
